El siguiente repositorio contiene el código implementado para El funcionamiento de un bot con Node.js y FastAPI así como una DB NoSQL para el funcionamiento del mismo_

### Pre-requisitos 📋

_En este repositorio específico se mantendrá La integración de FastAPI con el sistema de Node.js_ 

Para iniciar con La integración es necesario crear un entorno virtual con

* virtualenv
* FastAPI
* uvicorn
* openai 
* pyMongo

```
pip instala virtualenv
```
